# Credit Risk Prediction Model

# Project Overview
This project builds a machine learning model to predict whether a customer is likely to default on a loan.

# Objective
To support better loan approval decisions by identifying high-risk customers.

# Dataset
German Credit Dataset (1000 customers)

# Models Used
- Logistic Regression
- Random Forest

# Results
- ROC-AUC: ~0.82
- Accuracy: ~80%

# Key Insights
- High-risk customers can be identified with good accuracy
- Adjusting the decision threshold reduces financial risk
- Trade-off between loan approval and default risk

# Business Value
This model can help financial institutions:
- Reduce loan defaults
- Improve risk assessment
- Make data-driven lending decisions
